# Запуск приложения 

Инструкция для запуска приложения локально, с возможность просмотра демо на устройстве

## Подготовка

Необходимые утилиты, которые нужно установить предварительно

 - [Git](https://git-scm.com/install/)
 - [Node.js](https://nodejs.org/en/download)

 ### Установка Git (MacOS)

Самый просто способ – сделать это через Xcode Command Line Tools (способ из ссылки), то есть просто через Terminal. В терминале достаточно прописать команду:
```
git
```

Если git не установлен, то система предложит установить вам XCode, внутри которого и будел локальный git, с которым можно будет в дальнейшем взаимодействовать

### Установка Node.js (MacOS)

Эта утилита ставиться просто с официального сайта, проще всего это сделать через `.pkg` файл (простейшая Маковская установка). Главное правильно выбрать версию (с пометко LTC) и архитектуру:

- x64 под Intel
- ARM64 под Apple Silicon (M1, M2, M3, ...)

## 

Если всё установлено верно, то команды:
```
git --version
node --version
```
будут показывать соответствующий результат с номерами версий.

## Клонирование репозитория

Задача состоит в том, чтобы загрузить локально папку с файлами из данного репозитория

- Можно просто скачать архив с файлами

- Либо создать папку в произвольном месте, открыть эту её через терминал и с помощью команды ниже загрузить весь репозиторий
```
git clone https://github.com/Yehor-Somov/test-app-karta.git
```

## Загрузка пакетов

После загузки репозитория переходим внутрь папки test-app-karta с помощью команды:
```
cd test-app-karta
```

Ставим библиотеки командой:
```
npm install
```

Запускаем приложение командой:
```
npx expo start --tunnel
```

Как итог, получаем QR-код для сканирования, который нам понадобится далее


## Expo Go

В AppStore находим официальную апку [Expo Go](https://apps.apple.com/ua/app/expo-go/id982107779?l=ru)
Её достаточно просто загрузить, она будет служить как платформа для внутренней сборки проекта
После чего можно просто отсканировать QR-код (просто через приложение камеры, ибо в IPhone нет этой функции прям нативно в приложении, почему-то) и после небольшой сборки приложение будет доступно
